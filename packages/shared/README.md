# @bank-cc/shared

Shared types, validation, and utilities for the subcap calculator.

## Exports
- `@bank-cc/shared/types`: JSDoc typedefs.
- `@bank-cc/shared/validation`: Payload and settings validation helpers.
- `@bank-cc/shared/utils`: General helpers (parsing, normalization, IDs, base64 conversions).

## Usage
```js
import { parseAmount, arrayBufferToBase64 } from '@bank-cc/shared/utils';
```
