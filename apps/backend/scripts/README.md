# Backend Scripts

Helper scripts for backend validation. These are separate from the Jest test suite in `src/__tests__`.

## Shell
- `test-rate-limiting-integration.sh` - Rate limit integration checks
- `test-security.sh` - Security checks
- `test-validation-only.sh` - Validation-only test run
- `test-medium-security-fixes.sh` - Medium security fixes validation

## Node
- `test-rate-limiting.js` - Rate limiter behavior checks
- `test-security-implementation.js` - Security implementation checks
